# FormData.delete()

The **`delete()`** method of the `FormData` interface deletes a key and its value(s) from a `FormData` object.

## **Syntax**

```jsx
delete(name)
```

## **Parameters**

- `name`: The name of the key you want to delete.

## **Return value**

None (`undefined`).

## **Examples**

```jsx
formData.delete("username");
```
