# Model Queries

In Prisma, model queries are the specific queries you write to interact with your database models. These queries leverage Prisma Client to perform operations on the models defined in your Prisma schema. Prisma models are defined in the Prisma schema file (`schema.prisma`) and correspond to tables or collections in your database.
