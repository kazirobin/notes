# SVG Shapes

SVG shapes are **two-dimensional graphics defined using the Scalable Vector Graphics (SVG) format**. SVG is a vector image format that uses XML to store images as vectors, rather than pixels. SVG shapes are commonly used to display illustrations, charts, and graphics on websites.
